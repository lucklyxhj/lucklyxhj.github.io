冒泡排序简记

#笔记
昨天复习了一下最简单的排序算法冒泡排序，感觉还是有挺多东西在里面的，小记一下，作为巩固，贴上最简单的代码

cpp

BubbleSort(int a[], int n)
{
    for(int i=0;i<n;i++)
        for(int j=1;j<n-i;j++)
            if(a[j-1]>a[j])
            {
                int temp = a[j];
                a[j] = a[j-1];
                a[j-1] = temp;
            }
}

即使是最简单的排序算法还是有很多可以思考的问题

#优化的思路
1.如果某一次遍历没有发生交换，那么说明已经排完序，可以设定一个标志来检测这种情况，从而优化该算法，当然，时间复杂度没有改变
2.如果从某一个位置以后都已经排好序，并且比前面的数要大，则接下来的遍历只需要对前面的序列进行，故可以设置一个标志位对每次交换做记录，从而缩小遍历的范围

#总结
自然，这种优化聊胜于无，但是其思想值得我们借鉴，具体代码就不贴出来了
