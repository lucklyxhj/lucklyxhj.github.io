一个算法题

看到一个算法题，断断续续看了两遍才理解其本质，也是醉了
题目如下：

  1. 给定一数组a[N]，我们希望构造数组b [N]，其中b[j]=a[0]*a[1]…a[N-1] / a[j]，在构造过程中，不允许使用除法：
  2. 要求O（1）空间复杂度和O（n）的时间复杂度；
  3. 除遍历计数器与a[N]、b[N]外，不可使用新的变量（包括栈临时变量、堆空间和全局静态变量等）；

代码实现见MoreWindows Blog

下面贴一个比较优雅的实现方法，来源于其博客文章的第49条回复。

Cpp
void Example(int *a, int *b, int n)
{
    for(int i = 0; i < n; i++)//对数组b初始化
        b[i] = 1;
    for(int i = 1; i < n; i++)
    {
        b[i] *= a[0];//计算出元素b[i]的前半部分，并且利用a[0]来存储临时数据
        a[0] *= a[i];

        b[n-1-i] *= a[n-1];//计算元素b[i]的后半部分，并且利用a[n-1]来存储临时数据
        a[n-1] *= a[n-1-i];

    }
}

该段算法非常简洁，在博主的基础上减少了一个循环，特在此记录学习。
