前端笔记

#older IE bug
if an element and its container both set position as relative or absolute,
and initially rendered as none for display, once the status of container changed, even the element still got none in display, it will show up.
see in the details

#float and display-inline-block
use inline-block to get an effect like float, even can take the place of flaot.
see the code below:
style:

ul#display,
ul#display li {
    margin: 0;
    padding: 0;
}

ul#display li {
    display: inline-block;
    width:100px;
    min-height: 100px;
    background: #ccc;
    vertical-align: top;

    /* For IE 7*/
    zoom:1;
    *display: inline;
}

html:

<ul id="display">
    <li>Item one</li>
    <li>Item two</li>
    <li>Item three</li>
</ul>
