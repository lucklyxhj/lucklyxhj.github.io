插入排序的算法

#学习算法之一
开始看算法导论这本书，首先是插入排序算法，思想是 将无序数组向有序数组中逐个插入，第一步寻找插入位置，第二步位置以后的有序数组后移，插入该元素，挺简单的，放一个很优雅的写法

cpp

void InsertSortArray(int [], int n)
{
    for(int i=1;i<n;i++)//从第二个元素开始与前面排好序的数组作比较
    {
        int temp = arr[i];//标记用于交换
        int j = i - 1;    
        while(arr[j]>arr[i])//标记元素在j元素之前，则j元素后移一位，会覆盖i元素
        {
            arr[j+1]=arr[j];
            j--;
        }
        arr[j+1]=temp;//j最后自减一位了
    }
}
