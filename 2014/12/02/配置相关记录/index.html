#WAMP Server搭建
在win7中安装wamp server后只能打开原有网页，新的project报错：500 Invalid command RewriteEngine.
查看错误日志，.htaccess: Invalid command ‘RewriteEngine’, perhaps misspelled or defined by a module not included in the server configuration.
解决： RewriteEngine命令需要rewrite mod支持. 打开Apache的配置文件httpd.conf, 取消注释LoadModule rewrite_module modules/mod_rewrite.so.
完美解决
