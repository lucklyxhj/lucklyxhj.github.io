sublime text2 Note

#快捷键使用

  1. Ctrl + Shift + P打开命令面板
  2. Ctrl + Shift + V粘贴并格式化
  3. Ctrl + D选择单词，重复可增加选择下一个相同单词
  4. Ctrl + X删除当前行
  5. Ctrl + M跳转到对应括号
  6. Ctrl + R前往指定method
  7. Ctrl + Shift + M选中当前括号内内容，重复可选中括号
  8. Ctrl + /注释当前行
  9. Ctrl + Shift + /当前位置插入注释
  10. Ctrl + Alt + /块注释，并Focus到首行，写注释
  11. F11全屏
  12. Alt + F3选择所有相同的词
  13. Alt + .闭合标签
  14. Alt + Shift + Number分屏显示

#常用插件

  1. EmmetZendCoding的升级版，快捷键Tab
  2. Alignment代码对齐，选中几行，Ctrl + Alt + A对齐了
  3. Theme-Soda更换主题，在setting-user中添加“theme”: “Soda Dark.sublime-theme”注意逗号的问题
